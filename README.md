# Transporte Rio Lavayen - SystemTMS

Sistema de Gestión de Transporte y Logística (TMS) para Transporte Rio Lavayen.

## Estructura del Proyecto

- `backend/`: API y lógica de negocio.
- `frontend/`: Interfaz de usuario y componentes.
- `AUDIT_*.md`: Reportes de auditoría de diferentes módulos.
- `PLAN_DE_TRABAJO.md`: Hoja de ruta y progreso del proyecto.

## Desarrollo

1. Instalar dependencias: `npm install` (en backend y frontend)
2. Ejecutar localmente: `npm run dev`
